# Roadmap

## v0.1.0-beta — Akron Cruise Sprint (Current)

- [x] OpenDRIVE import from OSM (pure Python, no CARLA)
- [x] Procedural road mesh generation from XODR
- [x] Chaos Vehicle physics with tuning data asset
- [x] Cruise Sprint game mode (Loading → Countdown → Racing → Finished)
- [x] Checkpoint gates and route splines
- [x] Race scoring with penalties and clean-driving bonus
- [x] Ghost replay recorder + interpolated player
- [x] Per-route JSON leaderboards with medal thresholds
- [x] Persistent settings (video/audio/controls)
- [x] NeonHUD with speedometer, RPM, timer, checkpoint progress
- [x] Minimap + Compass widgets
- [x] Day/Night cycle
- [x] Traffic spawner with radius culling
- [x] Interactive tutorial system (5 steps)
- [x] Achievement system with JSON persistence (10 achievements)
- [x] Post-race stats screen
- [x] Developer console commands (cheats + debug)

## v0.2.0 — Visual Polish

- [ ] Cesium for Unreal / Google 3D Tiles integration
- [ ] Niagara FX (exhaust, tire smoke, sparks, rain)
- [ ] Material system for roads, buildings, environment
- [ ] UMG Blueprint binding for all widgets
- [ ] AkronWorld.umap level file with proper lighting
- [ ] Sound design pass (ambient city audio, UI sounds)

## v0.3.0 — Content Expansion

- [ ] Additional Akron routes (3-5 more cruise sprints)
- [ ] Vehicle selection screen
- [ ] Multiple vehicle classes (sedan, sports, truck)
- [ ] Photo mode
- [ ] Custom keybinding support

## v0.4.0 — Multiplayer Foundation

- [ ] LAN session hosting (Steam/UE subsystem)
- [ ] LAN join/discovery
- [ ] Synchronized race starts
- [ ] Live ghost racing (2+ players)

## v1.0 — Full Release

- [ ] Additional cities (semantic compiler generalization)
- [ ] Online leaderboards (backend service)
- [ ] Steamworks integration (achievements, friends, matchmaking)
- [ ] Windows installer (WiX/NSIS)
- [ ] Official launch trailer
